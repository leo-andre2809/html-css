Prática com formatação de texto

Negrito / Destaque

Podemos aplicar negrito a um trecho de texto utilizando a tag <b>, porém, de maneira não semântica.

Para utilizar o texto em destaque de maneira semântica, podemos utilizar a tag <strong>

Itálico / Ênfase

Para aplicar escrita itálica a um trecho de texto, de forma não semântica, podemos utilizar <i>

Agora, para darmos ênfase a um texto de forma semântica, utilizamos a tag <em>
