Revisando formatações de texto e listas

Algumas formatações de texto

  • Para textos fortes, em negrito, utilizamos <strong>
  • Para textos com ênfase, em itálico, utilizamos <em>
  • Para marcar o texto, utilizamos <mark>
  • Para textos com letras pequenas, utilizamos <small>
  • Para texto riscado ou deletado, utilizamos <del>
  • Para texto inserido, sublinhado, utilizamos <ins>
  • Para texto sobrescrito, utilizamos <sup>
  • Para texto subscrito, utilizamos <sub>
  • Para a leitura de códigos facilitada, podemos utilizar a tag <code>
  • Para citações, podemos utilizar a tag <q>
  • Para citações em bloco, podemos utilizar a tag <blockquote>
  • Para criar um pop-up com o significado de um abreviação, podemos utilizar a tag <abbr>

Criação de listas

As formatações de texto acima já contam com um tipo de lista; a lista não ordenada, representada pela tag <ul> (unordered list)

Em HTML podemos trabalhar com três tipos de listas: Listas não ordenadas (<ul>), listas ordenadas (<ol>) e listas de definição (<dt> e <dd>)
