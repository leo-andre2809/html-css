Principais formatações

Negrito / Destaque

Aqui temos o exemplo de um texto em negrito utilizando a tag <b>, que é uma tag não semântica

Para fazer um texto em negrito e dar destaque de forma semântica, podemos utilizar a tag <strong>


Itálico / Ênfase

Nesta frase, temos o exemplo de um texto itálico utilizando a tag não semântica <i>

Já aqui, estamos utilizando a tag semântica <em>, para dar ênfase ao nosso texto


Texto marcado

Podemos marcar um texto utilizando a tag <mark>


Texto pequeno

Para textos com letras menores, podem utilizar a tag <small>


Texto deletado / riscado

Para marcar um texto como riscado, ou deletado, podemos utilizar a tag <del>


Texto inserido / sublinhado

Para marcar um texto como inserido ou sublinhado, utiliamos a tag <ins>


Texto sobrescrito

Para criar um elemento em texto elevado aos outros, como em "X2", devemos utilizar a tag <sup>


Texto subscrito

E para fazermos com que os elemntos fiquem abaixo, como no exemplo "H20", devemos utilizar a tag <sub>
